# Инструкция по запуску

node version 20.12.2

1. npm install - установка зависимостей
2. npm start - запуск dev сервера

# Описание
Одностраничное React-приложение с единственной страницей Events. Архитектура построена по модульному принципу: функциональность разделена на независимые модули (страница, компоненты, хуки), что упрощает поддержку и масштабирование.
Используется TypeScript, обеспечивающий строгую типизацию, улучшенную читаемость кода и снижение количества ошибок на этапе разработки
